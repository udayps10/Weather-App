# Weather App 🌦️

A simple weather application built using HTML, CSS, and JavaScript.  
This app allows users to search for any city and view the current weather conditions.

---

## 🔗 Live Demo

https://udayps10.github.io/Weather-App

---

## ✨ Features

- Search weather by city name
- Displays temperature and weather description
- Real-time weather data
- Simple and responsive interface

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

---

## ⚙️ How It Works

The app sends a request to the OpenWeather API with the city entered by the user.  
The API returns weather data which is displayed on the page using JavaScript.

---

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Enter a city name to see the weather
